Simple Calculator

Below are some of the operations which can be performed :

  1. addition (+) = number1 + number2
  2. subtraction (-) = number1 - number2
  3. multiplication (*) = number1 * number2
  4. divison (/) = number1 / number2
  5. squareroot (Math.sqrt) = Math.sqrt(number)
  6. percentage (n1/n2*100) = number1 / number2 * 100

Note: The JS code is written in a separate file, open console to view the output :D
