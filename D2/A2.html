Methods for Strings and Arrays:

Note:This html file contains the explaination of methods and the examples of these methods are avaiable in js file.

Methods for Strings:

1.slice():

Extracts a part of a string and returns a new string.

2.replace():

Searches a string for a specified value, or a regular expression and returns a new string where the specified values are replaced.

3.repeat():

Returns a new string with a specified number of copies of existing string.

4.trim():

Removes whitespaces from both end of the string.

5.substring():

Extracts the characters from a string, between two specified indices.

6.includes():

Checks whether the string contains the specified string or characters.

7.concat():

Joins two or more strings and returns a new joined strings.

Methods for Arrays:

1.concat():

Joins two or more arrays and returns a copy of the joined arrays.

2.copyWithin():

Copies array elements within the array, to and from specified positions.

3.fill():

Fill the elements in an array with a static value.

4.reverse():

Reverses the order of the elements in an array.

5.slice():

Selects a part of an array and returns new array.

6.splice():

Adds or removes elements of an array.

7.sort():

Sorts the elements of an array.
