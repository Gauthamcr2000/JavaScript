Difference between var, let and const in Javascript:

1.var:

  • var is a keyword used in javascript.
  • var statement is used to declare a variable.
  • if the variable is not assigned to any value, it shows undefined while trying to print that variable.
  • you can assign the value to the variable using '='.
  • its not mandatory yo assign the value immediately, you can assign the value it before using that variable.
  • the variables declared using var keyword are function scoped, which means the variables can be used anywhere in the program.

2.let:

  • let is a keyword in javascript.
  • let is also used to declare variables.
  • the variable declared inside let is block scoped.
  • these variables declared using let, cannot be used outside the block.

3.const:

  • const is a keyword in javascript.
  • constant can be declared using const in javascript.
  • const is similar to let when it comes to block scope
  • once a constant is been declared, we cannot reassign it to a new value.
