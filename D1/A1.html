Various Methods In Console Function :

1.console.log():

Outputs a message to the console.

2.console.clear():

Clears the console.

3.console.error():

Outputs an error message to the console.

4.console.warn():

Outputs a warning message to the console.

5.console.time():

Starts a timer, can track the time taken for an operation which is performed.

6.console.table:

Displays tabular data as a table.
