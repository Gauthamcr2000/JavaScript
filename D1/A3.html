Datatypes In Javascript:

There are five types of primitive data types in javascript:

1.String:

Represents a sequence of characters.
Ex : "Welcome To Javascript!"

2.Number:

Represents numeric values.
Ex : 560

3.Boolean:

Represents a boolean value either true or false.
Ex : True or False

4.Undefined:

Represents undefined value.
Ex : Undefined

5.Null:

Represents null, that is no value at all.
Ex : Null

There are three types of non-primitive data types in javascript:

1.Object:

Represents instance through which we can access members.
Ex : {f1:"apple", f2:"orange", f3:"mango"}

2.Array:

Represnts group of similar values.
Ex : [5,10,15,20,25]

3.RegExp:

Represents regular expression.
Ex : /google/i
